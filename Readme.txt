Макрос для ANSYS Mechanical вычисляет эквивалентные напряжения в выбранных узлах сетки КЭ и записывает в файлы в соответсвие с количеством сочетания расчётных нагрузок.
При необходимости сравнения результатов расчёта и натурных испытаний образца узлы сетки КЭ выбираются исходя из количества контрольных точек определённых в методике натурных испытаний конструкции. Это позволяет оценить сходимость расчёта.
